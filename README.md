## \ud83c\udfaf Overview

This project demonstrates end-to-end M&A analysis capabilities through a realistic acquisition scenario: **Salesforce acquiring ServiceNow for $165B**. It includes:

- \u2705 **Complete DCF Valuation Model** with WACC calculation and terminal value
- \u2705 **Comparable Company Analysis** using 10 SaaS/enterprise software firms
- \u2705 **Precedent Transaction Analysis** of 8 recent M&A deals
- \u2705 **Synergy Modeling** with $4.3B in revenue and cost synergies
- \u2705 **Accretion/Dilution Analysis** showing +8.5% EPS accretion
- \u2705 **Professional Dashboard** with interactive charts and downloadable reports
